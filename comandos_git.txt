git init .          -> Cria um repo git na pasta atual
git status          -> Mostra o status do trabalho
git add nome_arquivo -> Adiciona arquivo para ser commitado
git commit -m "msg" -> Faz commit do que estava para ser commitado
git log -> Mostra todo o histórico de commits
git diff <nome arquivo> -> Mostra a diferença do arquivo atual pro último estado anterior desse mesmo arquivo. também pode ser usado git diff . pra mostrar de todos os arquivos.
